# Comparação de Algoritmos: Balanceamento de Carga em Firewalls

Este projeto implementa e compara dois algoritmos de otimização — **Simulated Annealing** e **Hill Climbing (Busca Local)** — aplicados ao problema de distribuição e balanceamento de carga entre Firewalls ativos.

## 🚀 Como acessar e rodar o projeto localmente

Siga os passos abaixo no terminal do seu computador para baixar e executar o código:

### 1. Clonar o repositório
Baixe uma cópia dos arquivos para a sua máquina:
```bash
git clone https://github.com
```

### 2. Acessar a pasta
Entre no diretório do projeto:
```bash
cd calculadora
```

### 3. Instalar as dependências
Este algoritmo utiliza bibliotecas externas para cálculos matemáticos e geração de gráficos de convergência. Instale-as executando:
```bash
pip install numpy matplotlib
```

### 4. Executar o script
Rode o código Python para visualizar os resultados comparativos no terminal e o gráfico de convergência:
```bash
python algoritmos.py
```
